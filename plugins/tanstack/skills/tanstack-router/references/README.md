# TanStack Router References

Use these files when `tanstack-router` is the right macro-area but the exact Router subdomain still needs to be narrowed.

- `routing-structure.md`: route tree, layouts, path params, and type registration.
- `navigation-and-search.md`: search params, URL state, links, and imperative navigation.
- `data-loading-and-ssr.md`: loaders, preload behavior, cache boundaries, and Router SSR.
- `auth-and-failures.md`: auth guards, redirects, not-found behavior, and route errors.
- `plugin-and-splitting.md`: router plugin wiring, generated routes, and lazy-route boundaries.
