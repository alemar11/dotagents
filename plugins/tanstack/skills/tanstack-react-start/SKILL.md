# TanStack React Start

Use this skill when the task is broadly about `@tanstack/react-start` as a React framework and is not yet narrowed to server functions, middleware, or deployment.

## Owns

- High-level React Start app setup.
- Framework-level structure and broad Start guidance.
- Deciding when a problem is Start-specific rather than plain Router.

## Does Not Own

- Server function details: use `tanstack-start-server-functions`.
- Middleware concerns: use `tanstack-start-middlewares`.
- Query + Router + Start coordination: use `tanstack-integration`.

## Workflow

1. Confirm the task is really about React Start as the framework surface.
2. Place the concern into the right Start subdomain before making code-level recommendations.
3. Keep React-specific Start advice separate from lower-level core runtime concerns.

## Verification

Verify against current TanStack React Start docs or installed first-party Intent skills when exact framework APIs matter.
