# TanStack Start Core

Use this skill when the task is about the Start core runtime model and does not fit a narrower React-specific or server-only sub-skill.

## Owns

- Start core runtime patterns.
- Framework behavior shared across multiple Start concerns.
- High-level Start constraints that are not just React wrapper details.

## Does Not Own

- React app ergonomics: use `tanstack-react-start`.
- Server-only runtime behavior: use `tanstack-start-server-core`.
- Deployment target specifics: use `tanstack-start-deployments`.

## Workflow

1. Confirm the task is about Start core behavior rather than one narrow feature.
2. Keep advice centered on runtime model and framework invariants.
3. Hand off to a narrower Start skill once the problem becomes specific.

## Verification

Verify against current TanStack Start core guidance when exact APIs or runtime behavior matter.
