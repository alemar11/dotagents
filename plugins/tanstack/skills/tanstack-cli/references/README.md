# TanStack CLI References

Use these files when `tanstack-cli` is the right macro-area but the exact CLI workflow still needs to be narrowed.

- `scaffolding.md`: `tanstack create` and new app bootstrap.
- `addons-existing-app.md`: `tanstack add` on an existing project.
- `ecosystem-integrations.md`: compare installable ecosystem options.
- `custom-addons-dev-watch.md`: build and iterate custom add-ons or templates.
- `docs-and-library-metadata.md`: query machine-readable docs and library metadata.
