# Custom Add-ons And Dev Watch

Use this guide when the task is about authoring or iterating custom TanStack CLI add-ons or templates.

Owns:
- add-on init, compile, and dev-watch loops
- local iteration mechanics for custom CLI surfaces
- separating authoring workflows from ordinary app-consumer commands

Escalate to:
- `tanstack-cli-maintain-custom-addons-dev-watch` for direct leaf-skill handling
- repo maintainer workflows only when the task is about this repository’s own plugin maintenance rather than runtime CLI usage
